## 변수 표기법
1. camel
- 첫 문자 소문자, 띄어쓰기 대신 대문자
- Java 변수, 함수
ex) userId

2. snake
- 모든 문자 소문자, 띄어쓰기 대신 ' _ '
- DB 컬럼에 사용
ex) user_id

3. pascal
- 첫 문자만 대문자
- Java Class
ex) User
